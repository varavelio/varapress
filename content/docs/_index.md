VaraPress is the official Zola theme for **Varavel** projects. It gives you everything you need to build professional documentation, blogs, and landing pages, all with the Varavel design system baked in.

## What You Get

- **Documentation layout** — Three-column layout with sidebar navigation, content area, and table of contents. Just like you're reading now.
- **Blog layout** Clean, minimal blog with tags and RSS support.
- **Landing pages** Hero sections, feature grids, calls to action, etc.

## Design System

VaraPress implements the [Varavel Design System](https://github.com/varavelio/brand):

- **Monochromatic base** with a 4-tier elevation system
- **Geist Sans** for UI text, **Geist Mono** for code
- **Dark mode** with `light`, `dark`, and `system` preferences
